    Auth

    Direct Subclass:

    EmailAuth, FacebookAuth, GithubAuth, GoogleAuth, LinkedinAuth, NoAuth

    Authenticator/passport strategy wrapper abstraction.

    I.e. it is the parent class of all auth classes.
